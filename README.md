# Ammu Will You Talk To Me? 💔

A fun and interactive React UI project with animations and heart-themed interactions.

## Features ✨

- **Broken Heart Background** 💔 → **Healed Heart** 💚
- **Large Interactive Text** asking "Ammu, will you not talk to me from now??"
- **Yes Button** [Yes 💔] - Escapes your cursor (can't be clicked)
- **No Button** - Clickable with heart icon that changes to green when clicked
- **Smooth Animations** with Tailwind CSS
- **Sweet Message** after interaction: "I'll talk, Ronaldo! 💚✨"

## Installation 🚀

```bash
# Install dependencies
npm install

# Start the development server
npm start
```

The app will open at `http://localhost:3000`

## How It Works 🎯

1. **Hover over "Yes"** → Button runs away from your cursor 😄
2. **Click "No"** → Heart heals and message shows "I'll talk, Ronaldo!" 💚
3. **Background updates** → Broken heart becomes a healthy green heart
4. **Happy ending** → "You're the best! 🥰❤️"

## Technologies Used 🛠️

- **React** - UI framework
- **Tailwind CSS** - Styling
- **React Icons** - Beautiful icons
- **CSS Animations** - Smooth transitions

## File Structure 📁

```
src/
├── App.jsx          # Main component with logic
├── App.css          # Custom animations
├── index.jsx        # React entry point
└── index.css        # Global styles
public/
└── index.html       # HTML template
```

## Customization 🎨

- Change colors in `App.jsx` (Tailwind classes)
- Modify animation speeds in `App.css`
- Update messages in the JSX
- Add more emojis or effects

## Build for Production 📦

```bash
npm run build
```
